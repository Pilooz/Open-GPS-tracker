Open GPS-tracker

## Installation

### Localhost ssl installation

```bash
cd docker/conf/
openssl req -x509 -nodes -days 365 -newkey rsa:2048 -keyout nginx-selfsigned.key -out nginx-selfsigned.crt
```
Launch docker containers at the project's root :

```bash
docker-compose up -d
```
